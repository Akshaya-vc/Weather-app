# Weather-app
Displays the current weather in a given location
## Built using
* Node.js
* Express
* Weather API - [Dark Sky](https://darksky.net/forecast/40.7127,-74.0059/us12/en)
* Geolocation API - [Map Box](https://www.mapbox.com/)
